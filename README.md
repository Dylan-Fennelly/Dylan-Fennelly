# 👋 About Me

Hi, I’m **Dylan** — a software developer and game developer currently completing an **MSc in Computing (Games & Extended Reality)** while working professionally as a **Junior Software Developer**.

My work sits at the intersection of **software engineering, game development, and interactive systems**. I enjoy building systems that combine strong technical foundations with experimental ideas, particularly in **real-time environments and simulation**.

Professionally, I work on maintaining and modernising a large **industrial web application platform**, primarily using **.NET and AngularJS**, where I focus on improving **reliability, telemetry, testing infrastructure, and performance** across legacy systems.

Alongside that, I build **games and experimental interactive projects** using **Unreal Engine (C++ & Blueprints)**.  
My current project explores **machine perception, LiDAR-style scanning, and non-Euclidean spaces** to create unconventional player experiences.
